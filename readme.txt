Want to use specific jQuery or Javascript for each page, post or product? Just install it.

== Description ==
This plugin will add jQuery box under each post, page or custom post where you can write your jQuery or JavaScript code that will run on only that specific page.

== Installation ==
1. Go to plugins in your dashboard and select \'add new\'
2. Search for \'jQuery in Posts Pages\', install it and activate
3. Go to any page or post and edit
4. Here you will see a box for jQuery code, write your jQuery here and update that post/page
5. If you didn\'t find any meta box as defined above, click on screen options and then check the checkbox labeled \"jQuery or JavaScript\"
